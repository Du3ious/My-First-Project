CSS INTRODUCTION

CSS (Cascading Style Sheets) is how you bring styling and formatting into HTML code.

Although you are able to write CSS inline with HTML code, it is better to link CSS styling through an external style sheet.

To link an external style sheet, you use the HTML <link> element in the head of the sheet.

The <link> element is self closing (no closing tag) and uses three attributes:

  • rel - specifies the relationship between the current page and the page it is been linked to. For CSS links, the relationship should be "stylesheet"

  • type - specifies the type of file that is been linked to. This should be "text/css" (I don't know why yet)

  • href - provides the URL for the link. In this case to the "*.css" sheet

CSS code is classified as 'Rules'. Each rule has a selector, which specifies with HTML elements to style.

After the selector, there are properties and values that distinguish exactly how the selected code will be styled.

eg. h1 { font-family: Georgia, serif; }

  • font-family - selects the font to be displayed. Any fonts listed after the first font are used in order as 'fallback fonts'. It is always a good idea to include a web safe fallback font.

  • font-size - sets the font size using pixels, em, rem, and percentages. (Refer to this explanation for more info)

  • text-align - sets the text alignment

  • color - sets the text color using names (eg. red), RGB (eg. rgb(128,15,128)), or Hexadecimal colour (#RRGGBB) (Refer to here for a colour picker)

  • .* - is a class selector and specifies a parent class in the HTML code to syle

  • #* - is an id selector and specifies a parent identifier in the HTML code to style
