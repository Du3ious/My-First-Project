HTML INTRODUCTION

HTML is composed of elements. Elements are composed of an opening and closing tag, with the content placed inbetween. Closing tags always have a foward slash (/) in the them.

Elements have a heirachy, or family lineage, which is used to group sections of code. The highest parent element is HTML. head and body are the child elemnts of HTML, and are siblings. All other elements are children of either head or body.

When writing a HTML doc, the first line of code should be the document type declaration, which tells the browser that the code is in HTML. The document type declaration looks like <!DOCTYPE html>

After the document type declaration, to start the structure of the code to read, HTML tags need to be added. This will instruct the browser to generate all code within the tags as HTML. The HTML tag is <html>

A HTML doc consists of the head, which contains the metadata about the webpage, and the body, which contains the code which will be displayed on the page itself.

LISTS

There are two sorts of lists in HTML: Unordered, and Ordered lists.

To create an unordered list in HTML, use the ul element to create the list, and li for each item for the list to contain as a dotpoint

To create an ordered list, use the ol element to create the list, and li for each item for the list to contain as a numerically ordered list.

When creating a list within other text, close the text tag first (eg <p>) and open a new text code when the list is finished.

ATTRIBUTES

Attributes are added to an opening tag and are used to distinguish tags. They can be used for a variety of things, from providing information, through to changing the style of the content. Most attributes require a defined value (="*"), however attributes which rely on a true/false indication (boolean values) do not.

ATTRIBUTE EXAMPLES

  • id - one use of id is to give an idenitifier to an element, and is useful if you need to distinguish between frequently used elements such as 'div'. It can be used on almost any element to give it an identifier.

  • class - similar to id, however where id is used to distinguish individual specific elements, class is used to identify elements that are all alike. Note = if both 'class' and 'id' are used, CSS will use the 'id' (Refer to this explanation if stuck)

  • src - links to a source material such as an image or a video

  • controls - when added to a video tag, allows the use of video controls. (Refer to the VIDEOS chapter)

  • href - gives a hyperlink a destination. (Refer to the HYPERLINKS chapter)

  • target - tells a hyperlink how to open (new tab, new window, same window, etc. (Refer to HYPERLINKS chapter)

  • scope - indicates column or row for a table heading (Refer to the TABLES chapter)

IMAGES

To add an image, use the img tag. The img tag is self closing and does not require a closing tag.

The img tag requires a source, which is determined by src="*"

All images should have alternative text, or image descripter, written within the code. This is useful for SEO, the visually impaired, and leaves a description if the image fails to load. This uses the alt"*" tag.

VIDEOS

Videos work similarily to images, however it does use a closing tag. To add a video, use the video tag. Include the source, width and height, and video controls within the opening tag. Any text written between the opening and closing tags will be displayed if the video fails to load properly.

To add the source, use the src="*" tag, the width and height are set with the width="*" and height="*" tags, and the allowance of video controls are set with the controls tag. The controls tag is a kind of boolean value (true/false), and doesn't need a value (eg. ="*").

HYPERLINKS

TABLES

Tables are created using the <table> tag, and can be used to store and retrieve data. Once a table has been created, you must create the individual rows, and then fill each row with cells of data.

Rows are created with the <tr> tag, with each opening and closing </tr> creating one row.

Within each row, cells can be added using the <td> tag. Any text input between the <td> tags is displayed in the cell.

By repeating the process of creating rows and cells, a table is formed.

To create titles for each row and column, the <th> tag is used, along with the attribute scope="col" or scope="row". The scope indicates whether the title is for a row or a column.

If any cell data needs to cover more than one cell, the attributes colspan="*" or rowspan="*" are used.

If a table is growing large, the data can be seperated (eg. the data from the headings) using the <tbody>, <thead>, and <tfoot> tags.
